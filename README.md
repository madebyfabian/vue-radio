# vue-radio
A minimal PWA for listening to your favorite radio station out of 100.000+. Uses the TuneIn API.

<br>
<hr>
<br>

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
